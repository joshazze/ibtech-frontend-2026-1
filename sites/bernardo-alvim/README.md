# Cartao de Visita Pessoal - IbTech

Projeto 01 da Trilha Frontend: uma pagina pessoal feita com HTML, CSS e JavaScript puros.

## Requisitos atendidos

- HTML semantico com `header`, `nav`, `main`, `section`, `article`, `figure`, `address` e `footer`.
- CSS separado com variaveis, responsividade e estados de interacao.
- JavaScript separado com quatro interacoes:
  - menu responsivo;
  - alternancia de tema claro/escuro;
  - botao para copiar email;
  - seletor criativo de foco do semestre.
- Conteudo pessoal com nome, sobre, fotos, video, projetos, contato e elemento criativo.

## Como visualizar

Abra o arquivo `index.html` no navegador.
