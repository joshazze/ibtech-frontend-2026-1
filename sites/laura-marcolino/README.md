# Projeto 01 — Cartão de Visita Pessoal

Nome: Laura Marcolino Ramos  
Turma: IbTech Frontend 2026.1

## Descrição

Este projeto é uma página pessoal desenvolvida como cartão de visita digital. A página apresenta informações sobre mim, minha trajetória, projetos, experiências, redes sociais e formas de contato.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Tema claro e escuro
- Botão para copiar e-mail
- Cards de projetos e experiências
- Elemento criativo com estrelas animadas
- Layout responsivo

## Como rodar o projeto

Para abrir o projeto, basta baixar os arquivos e abrir o arquivo `index.html` no navegador.
