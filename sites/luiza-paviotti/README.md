# Cartão de visitas - Luiza Paviotti Pendio - IBTECH 2026.1

## Descrição:
Desenvolvido em HTML e CSS com o objeito de criar um cartão de visitas digital/apresentaçao pessoal. A página reúne informações sobre minha tragetória até o momento, acedêmicas, habilidades, interesses e forma de contato, de maneira organizada. 
A estilização do site foi pensada para o menu ser um céu estrelado (estrelas piscando), com o céu em tons de azul, roxo e rosa, atraves da criatividade. Os efeitos de brilho, animação e o fundo em degradêforam usados para criar uma imagem moderna e delicada, refletindo meu interessse pela área de tecnologia e minha personalidade. 

## Tecnologias Utilizadas:
- HTML
- CSS

## Como executar
Abrir o arquivo index.html em qualquer navegador.

## Funcionalidades 
- Menu de navegação
- Efeitos visuais com animações
- Layout personalizado
- Sessão de habilidades
- Cards de contato
- Desing responsivo básico
