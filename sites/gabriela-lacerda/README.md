# Sobre

O projeto foi desenvolvido como um portfólio pessoal com visual minimalista e moderno, misturando tecnologia, design e elementos inspirados em cybersegurança.

A proposta foi criar uma interface elegante e interativa, mas ao mesmo tempo limpa e organizada, funcionando como um cartão de visita digital.

Além da parte visual, o foco também foi trabalhar interações em JavaScript puro, responsividade e uma estrutura semântica utilizando apenas HTML, CSS e JavaScript vanilla.

# Decisões de projeto

## Visual minimalista

O projeto foi desenvolvido com foco em uma estética mais clean e sofisticada, evitando elementos exagerados e priorizando contraste, espaçamento e tipografia.

## Interatividade sem frameworks

Todos os efeitos visuais e interações foram feitos utilizando apenas JavaScript puro, sem bibliotecas externas.

## Responsividade

A interface foi pensada para funcionar tanto em desktop quanto em dispositivos móveis, utilizando Grid, Flexbox e media queries.

## Estrutura organizada

O projeto foi separado em arquivos simples e organizados para facilitar manutenção e entendimento do código.
