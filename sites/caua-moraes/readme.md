# Cauã Moraes — Cloud & DevOps

**Autor:** Cauã Moraes  
**Turma:** IbTech — Trilha Frontend (2026.1)

## Descrição
Página pessoal de apresentação atuando como cartão de visita web e portfólio inicial. O projeto adota uma estética inspirada nos designs da Apple e apresenta um easter egg de Counter-Strike como elemento criativo.

## Tecnologias Utilizadas
*   **HTML5:** Estruturação semântica e acessibilidade.
*   **CSS3:** Variáveis nativas, Flexbox, Grid, design responsivo (Mobile First) e temas claro/escuro.
*   **JavaScript (Vanilla):** IntersectionObserver, Clipboard API e manipulação do DOM. Sem uso de frameworks ou bibliotecas externas.

## Como Executar
O projeto não requer build ou dependências. Basta clonar o repositório e abrir o arquivo `index.html` no seu navegador de preferência.
